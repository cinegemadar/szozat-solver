# Szozat & wordle solver

Hosted on [Heroku](https://polar-fortress-90021.herokuapp.com/docs)

## Components
- Core
- CLI
- Server

## install
```bash
python3 -m pip install -r requirements.txt
python3 -m pip install uvicorn
```

## run
```bash
uvicorn solver_server:app --reload
```

## test
